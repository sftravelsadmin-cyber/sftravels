# SF Travels - Vehicle Fleet Images

Please upload the fleet/vehicle images into this folder (`public/images/vehicles/`). 

### Important Guidelines:
1. **Format**: All images should be in **WebP** format (`.webp`) for maximum performance and premium loading speed.
2. **Naming**: File names must match the exact names below (all lowercase, hyphenated).
3. **Dimensions / Style**: For a clean, uniform look, try to use high-quality images with transparent backgrounds or consistent studio backgrounds, and crop/size them uniformly.
4. **Size**: The recommended size is **`960 x 600` pixels** (or a **16:10 aspect ratio**). 
   - *Minimum*: `800 x 500` px (for clean display on standard screens).
   - *Maximum*: `1200 x 750` px (anything larger will impact page load speeds).
